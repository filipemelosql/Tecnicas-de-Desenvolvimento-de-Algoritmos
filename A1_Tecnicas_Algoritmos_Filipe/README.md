# Avaliação A1 – Técnicas de Desenvolvimento de Algoritmos

Repositório criado para entrega da Avaliação A1 da disciplina Técnicas de Desenvolvimento de Algoritmos (UDF).

## Estrutura do Repositório

Cada pasta representa um conteúdo trabalhado em aula:

- `condicional/` → Estruturas Condicionais  
  - `condicional_idade.py`: simula um sistema de verificação de idade para entrada em eventos.

- `repeticao/` → Estruturas de Repetição  
  - `repeticao_pares.py`: exibe os números pares de 1 a 100 utilizando os laços `for` e `while`.

- `listas/` → Listas  
  - `lista_alunos.py`: permite o cadastro dinâmico de nomes de alunos em uma lista.

- `dicionarios/` → Dicionários  
  - `dicionario_produtos.py`: cadastra produtos utilizando um dicionário com nome e preço.

## Como Executar os Programas

1. Certifique-se de ter o Python 3 instalado.
2. Abra o terminal (ou prompt de comando) na pasta do repositório.
3. Acesse a pasta do exercício desejado. Exemplo:

```bash
cd condicional
python condicional_idade.py
```

Ou:

```bash
cd repeticao
python repeticao_pares.py
```

## Exemplos Gerais de Uso

- Em `condicional_idade.py`, informe uma idade qualquer (por exemplo, 17, 25, 65) e observe as mensagens específicas para cada faixa etária.
- Em `repeticao_pares.py`, serão impressos na tela os números pares de 1 a 100 duas vezes: um bloco usando `for` e outro usando `while`.
- Em `lista_alunos.py`, digite vários nomes de alunos e depois digite `sair` para finalizar e ver a lista completa.
- Em `dicionario_produtos.py`, informe o nome e o preço de cada produto e digite `sair` para encerrar e visualizar todos os produtos cadastrados.
