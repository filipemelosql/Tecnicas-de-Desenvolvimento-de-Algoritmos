# Sistema de Gestão Básica

Projeto acadêmico bem direto ao ponto, mas organizado como se fosse um sisteminha real.
A ideia é juntar em um lugar só os principais conteúdos da disciplina:

- Estruturas Condicionais (if, elif, else)
- Estruturas de Repetição (for, while)
- Listas
- Dicionários

Tudo isso amarrado em um único menu, no arquivo `main.py`.

## Estrutura do Projeto

```
Sistema_Gestao_Com_Comentarios/
├── main.py
├── verificacao_idade/
│   └── idade.py
├── analise_numeros/
│   └── pares.py
├── cadastro_alunos/
│   └── alunos.py
└── cadastro_produtos/
    └── produtos.py
```

## Como executar (no computador)

1. Garanta que você tem o Python instalado.
2. Entre na pasta do projeto pelo terminal, algo como:
   ```bash
   cd Sistema_Gestao_Com_Comentarios
   ```
3. Rode o arquivo principal:
   ```bash
   python main.py
   ```

## Como executar no Google Colab

Se você estiver no Colab:

1. Envie os arquivos do projeto ou recrie a estrutura com código.
2. No final, rode:
   ```python
   !python main.py
   ```

O sistema vai abrir o menu no console e você escolhe as opções digitando os números.
